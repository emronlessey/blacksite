Escape Room Game

MINI BLACKSITE

How to Play

You are trapped inside a blacksite facility and must solve 3 puzzles before time runs out.

Rules

  • Press Start Game to begin.
  • You have 2 minutes to escape.
  • Complete all 3 puzzles in order.
  • You can use up to 3 hints.
  • If time reaches zero, lockdown begins.
  • In Puzzle 3, only one button is correct.

Puzzles

  • Puzzle 1: Memorise the code and type it back.
  • Puzzle 2: Wait for the signal, then submit at the right time.
  • Puzzle 3: Find the safe button without too many wrong tries.
